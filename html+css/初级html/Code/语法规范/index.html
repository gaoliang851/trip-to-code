兄弟对决

据每日电讯报报道，总统府外的街道上挤满了悼念者。值得一提的是，部分狂热粉丝与警察爆发冲突。航拍画面显示，双方推搡互骂，警察一度被围攻。据报道，随着排队进场时间拉长，前来告别的球迷情绪愈发激动，粉丝试图破坏围栏，推开警察溜进总统府。报道称，暴力骚乱已造成多人受伤。
